Documentation

Each step has it's own custom options, so you can style every single step different, clone it, highlight it and much more. There are 4 tour types so some options dont work in every type, you can see which type has which options available. To make things easy we included a tour build form, which helps you build a good tour in seconds. You also can start the tour on pageload and go to a new page and continue the tour there.

You can build pretty cool and creative tour with the Power Tour plugin.

Install

Add the following file's to your <head> section.

<!-- PowerTour stylesheet -->
<link rel="stylesheet" href="css/powertour.css"/>
<!-- jQuery core -->
<script src="js/jquery-1.10.2.min.js"></script>
<!-- PowerTour plugin -->
<script src="js/powertour.1.4.min.js"></script>

Plugin setup

How to use the plugin.

$(document).ready(function($){
    $('#starttour').powerTour({
        tourType: 'auto',
        overlayOpacity: 0.5,
        showTimeControls: true,
        showTimer: true,
        timeButtonOrder: '%prev% %stop% %pause% %play% %next%',
        stepButtonOrder: '%prev% %next% %stop%',
        timerLabel: 'Next step in: ',			                    
        effectSpeed: 200,
        travelSpeed: 400,
        easingEffect: 'linear',
        easyCancel: true,
        animated: true,
        keyboardNavigation: true,
        runOnLoad: false,
        runOnLoadDelay: 2000,
        exitUrl: '',
        onStart: function(){},
        onFinish: function(){},
        onStop: function(){},	
        onCancel: function(){},
        step:[
              {
		hookTo: '#step-trigger',
		content: '#box-trigger',
                cloneTo: '',
                width: 400,
                arrowPosition: 'sc',
                showArrow: false,
                offsetY: 0,
                offsetX: 0,
                prevLabel: 'Prev',
                nextLabel: 'Next',
                stopLabel: 'Stop',
                center: 'hook',						  
                customClassStep: '',
                customClassHook: '',
                highlight: true,
                highlightElements: '',
                time: '00:05',
                onShowStep: function(){},
                onHideStep: function(){}
              },
              {
              // next step
              }

});

Usage plugin main options

How to use the plugin main options.

Plugin options Type Default Desctription Tour type Since
tourType string auto This is the tour type, you can choose between 4 types 'auto' which will automaticly go to the next step, 'step' which wil go to the next step with a click of a button, 'showAll' which will show all steps at once and 'tooltip', which wil act as an tooltip once you hover the hook.   1.0
overlayOpacity number(float) 0.5 You can set the overlay opacity of the highlight option. auto/step/showAll 1.0
showTimeControls boolean auto The tour type 'auto' lets you use an small control panel with prev, next, pause, stop and play buttons, which you can use during the tour. auto 1.0
showTimer boolean true You can show an timer which will tell when the next step will start. auto 1.0
timeButtonOrder string %prev% %stop% %pause% %play% %next% Choose the button order of the tour type 'auto' control panel. auto 1.0
stepButtonOrder string %prev% %next% %stop% Choose the button order of the buttons that are located in the step it self. auto 1.0
timerLabel string Next step in: The text that is used for the 'showTimer' option. auto 1.0
effectSpeed number(integer) 200 The speed of the effect(show/hide). auto/step 1.0
travelSpeed number(integer) 400 The speed between going from on step to the next step. auto/step 1.0
easingEffect string linear The kind of effect thats used between the changing of the steps. Default is 'linear', but you can use more easing effect if you use the jQuery UI core. Easing demo's. auto/step 1.0
easyCancel boolean true Easy cancel if a fast way to cancel a tour, the tour will stop immediately if a user clicks outside the step. auto/step/showAll 1.0
animated boolean true You can choose if the tour is animated. auto/step 1.0
animatedOnEnd boolean true Once the tour is done animated back to the top. auto/step 1.0
keyboardNavigation boolean true The tour type 'auto' and 'step' do support keyboard keys like space, esc, arrow key left and arrow key right. See the demo's for more. auto/step 1.0
runOnLoad boolean false The tour runs when a user clicks the tour button, but you can do a tour that runs on several pages, if you set this to true the tour will run if the pages get's loaded, so you can start a tour with a click and enter a exitUrl so that you can continue on a different page. You dont have to set this to true, if the last page has an exitUrl(the tour uses a parameter to start the tour). NOTICE: you still have to use an exciting id/class for the trigger in order to get the tour working. auto/step/showAll 1.1
runOnLoadDelay number(integer) 2000 If you use the 'runOnLoad' option you can give it an delay. auto/step/showAll 1.1
exitUrl string   If you enter a valid exitUrl the tour will go to this url once the tour is done, this way you can continue the tour on a different page. Keep this empty if you dont want to use it! You can use parameters in the url. auto/step/showAll 1.1
step array [] These are the steps that you must use for the tour. See the doc below on how to use it. auto/step/showAll/tooltip 1.0
onStart function function(){} This is an callback fuction that runs once the tour starts. auto/step/showAll 1.0
onFinish function function(){} This is an callback fuction that runs once the tour is finished. auto/step/showAll 1.0
onStop function function(){} This is an callback fuction that runs once the tour stops by using an stop button. auto 1.0
onCancel function function(){} This is an callback fuction thats runs if the tour gets canceled with the 'easyCancel' option or by using the ESC key. auto/step/showAll 1.0

Usage step options

How to use the tour steps. Every step has there own options, as you can see in the demo.

Plugin options Type Default Desctription Tour type Since
hookTo string   This is the id/class of the part that you want to highlight. Try to use only id's. You must include the hastag or dot like '.classname' or '#idname'. auto/step/showAll/tooltip 1.0
content string   This is the content thats used in the step, you can use any kind of content you want, even HTML. You must include the hastag or dot like '.classname' or '#idname'. auto/step/showAll/tooltip 1.0
cloneTo string   You can clone the step to a new/extra hook, this way you dont have to use the same step after each other. The 'center' option works only for the main step. auto/step/showAll/tooltip 1.0
width number(integer) 300 Set the width of the step. auto/step/showAll/tooltip 1.0
arrowPosition string rt Set the arrow postion, this is the position where to step is connected to the hook. See the doc below for available positions. auto/step/showAll/tooltip 1.0
showArrow boolean true If you dont want to show an arrow use this option to switch it off. auto/step/showAll/tooltip 1.0
offsetY number(integer) 0 Offset of the step, this changes the vertically postion of the step, can be used with positive and negative values. auto/step/showAll/tooltip 1.0
offsetX number(integer) 0 Offset of the step, this changes the horizontal postion of the step, can be used with positive and negative values. auto/step/showAll/tooltip 1.0
prevLabel string nbsp; The label of the previous button. Notice that if you dont give the button any text it will not be visible. step 1.0
nextLabel string nbsp; The label of the next button. Notice that if you dont give the button any text it will not be visible. step 1.0
stopLabel string nbsp; The label of the stop button. Notice that if you dont give the button any text it will not be visible. step/auto/showAll 1.0
center string hook Choose the center the hook or the step. auto/step 1.0
customClassStep string   You can give the step and custom class, this is used to style the step. auto/step/showAll/tooltip 1.0
customClassHook string   Want to change something on the hook once it's highlighted, just use a custom class for this. auto/step/showAll/tooltip 1.0
highlight boolean false You can highlight the hook and step once it's active. auto/step/show 1.0
highlightElements string   You can highlight extra elements. auto/step/showAll/tooltip 1.0
time time format 01:00 The tour type 'auto' runs on a set time, you can use up to an hour for every step. Notice that you use the right time format. auto 1.0
onShowStep function function(){} This is an callback fuction. auto/step 1.0
onHideStep function function(){} This is an callback fuction. auto/step 1.0

Globally And Per-Call Overridable Options

This plugin has per-call(as the example above shows) and globally overridable options. Globally options can be used to override the default options, this is best used if you want to override the plugin options by default, this will prevent any issues with the plugin, as you dont have to change any thing in the plugin core it self. How to use: $.fn.pluginName.key = 'value';
